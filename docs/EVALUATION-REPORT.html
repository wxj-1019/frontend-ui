1.2 违规项目

🔴 F-01: 禁止使用 Emoji 首页 page.tsx 第 19-42 行的 categories 数组使用了 ✦、⚡、◆、◎ 作为图标。根据 ANTI-EMOJI POLICY，应使用 Phosphor 或 Radix 图标库替代。
🔴 F-02: 使用 Inter 字体 globals.css 将 Inter 设为 --font-sans。规范明确禁止使用 Inter，应替换为 Geist、Outfit 或 Satoshi。
🔴 F-03: 自定义光标 CursorEffect.tsx 实现了全站自定义光标效果（GSAP 驱动）。规范明确禁止自定义光标（Forbidden Patterns > Visual）。
🔴 F-04: 多 accent 色 (5 个) 项目使用 5 个强调色: Cyan、Magenta、Violet、Success、Warning。规范要求 ≤1 accent。可保留 Cyan 作为唯一 accent，其他收归为语义色（Success/Error）。
🟡 W-01: 霓虹发光效果 Hero 区域使用 blur-[150px]、bg-[var(--color-accent-cyan)]/15 等模糊发光。规范禁止 Neon glows（Forbidden Patterns）。
🟡 W-02: 首页统计数据不准确 stats 数组声称 "130+ 动画组件"、"5 组件分类"，实际只有 34 个组件。数字虚高有损信任。
🟡 W-03: 纯黑背景 --color-bg-primary: #0A0A0F 接近纯黑。规范禁止纯黑 (#000) 作为背景色。
🟡 W-04: 过度饱和的 accent 颜色 --color-accent-cyan: #00F5FF 饱和度过高。规范要求 saturation < 80%。

1.3 通过项
